  • git config --global user.email blabal@bla
  • git init - створює новий локальний репозиторій Git в поточній директорії.
  • git status - використовується для перевірки статусу файлів в робочій директорії та індексі в Git репозиторії.
  • git add - додає файл або зміни в файл до індексу Git репозиторію.
  • git commit - зберігає зміни в репозиторії Git.
  • git log - виводить список історії комітів у вашому Git репозиторії.
  • git checkout - використовується для перемикання між гілками та комітами в Git репозиторії.
  • git branch - використовується для перегляду, створення та видалення гілок в Git репозиторії.
  • git push - використовується для відправки локальних змін до віддаленого репозиторію Git.
  • git pull - використовується для отримання та об'єднання змін з віддаленого репозиторію Git до вашого локального репозиторію.
  • git merge - використовується для об'єднання змін з іншої гілки в поточну гілку.
  • git clone - використовується для створення копії віддаленого репозиторію Git на вашому локальному комп'ютері.

Файл .gitignore - це конфігураційний файл, який використовується системою контролю версій Git для вказівки файлів або каталогів, які не повинні враховуватись Git при відстеженні змін у репозиторії. Зазвичай цей файл розміщується в кореневому каталозі репозиторію Git і містить список файлів або каталогів, які повинні бути проігноровані.

Advanced:

  • git revert - це команда в системі контролю версій Git, яка дозволяє відмінити зміни, зроблені в попередніх комітах.
  • git reset - це команда в системі контролю версій Git, яка дозволяє скинути стан репозиторію до певного коміту, видаливши всі зміни, зроблені після цього коміту.
  • git rebase - це команда в системі контролю версій Git, яка дозволяє перебудувати гілку комітів на іншу базову гілку
  • git cherry-pick - це команда в системі контролю версій Git, яка дозволяє взяти один або декілька комітів з іншої гілки та застосувати їх зміни до поточної гілки.
  • git stash - це команда в системі контролю версій Git, яка дозволяє зберегти незбережені зміни в стек "збережень" (stash)
